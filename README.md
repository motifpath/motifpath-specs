# motifpath-specs

Product specifications, architecture decision records (ADRs), and design documents for the Motifpath project.

## Structure

```
specs/        Feature and product specifications
adr/          Architecture Decision Records
design/       System and UI design docs
api/          API contracts and schemas
```

## Contributing

- Open a PR for any new spec or ADR
- Use the templates in each directory as a starting point
- Link related specs and ADRs to keep context connected
